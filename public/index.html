Reading

03

Codex CLI + ChatGPT subscription

Codex виконує делеговані задачі та незалежні перевірки.

Reading

Codex встановлюється у user-space. Login запускається без OPENAI_API_KEY і має завершитися методом ChatGPT або enterprise access token, а не Platform API billing.

04

Claude → Codex bridge plugin

Офіційний OpenAI marketplace та codex:codex-rescue subagent.

Reading

Router додає openai/codex-plugin-cc через Claude CLI, встановлює plugin у user scope та перевіряє agent/runtime файли. Cache plugin-а вручну не редагується.

05

Routing policy + telemetry

Моделі, effort, plan audit, backups і локальна історія рішень.

Reading

Apply створює backup, записує глобальну routing policy та додає лише власні hooks. Інші Claude settings і hooks зберігаються.

06

Перевірка з’єднання

Спочатку без токенів, потім — опційний мінімальний live handshake.

Reading

Self-check перевіряє plugin runtime, agent manifest, Codex binary та subscription auth без виклику моделі. Live handshake запускає один короткий Codex turn і не читає та не змінює файли.
